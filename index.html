display: flex

A propriedade display flex deve ser a primeira a ser declarada no CSS, a partir dela é que o CSS vai entender que precisa estilizar o conteúdo do container com a Caixa Flexivel (Flexbox).

flex-direction

A flex-directionpropriedade especifica a direção dos itens flexíveis. Nota: Se o elemento não for um item flexível, a flex-directionpropriedade não terá efeito.

Align-itens

A align-itemspropriedade especifica o alinhamento padrão para itens dentro de um flexbox ou contêiner de grade. Para que esta propriedade tenha algum efeito de alinhamento, os itens precisam de espaço disponível ao seu redor na direção apropriada.
